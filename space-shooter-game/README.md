\# 太空射击游戏



一个简单的单文件Python太空射击游戏，使用Pygame库开发。



\## 游戏说明



\- 左右方向键：控制飞船移动

\- 空格键：发射子弹

\- 射击红色敌人获得分数

\- 被敌人撞到游戏结束

\- 游戏结束后按R键重新开始，按ESC键退出
